<h1 align="center">Todos smartphone app with SSL</h1>

| Layer | Technology |
| ------------- | ------------- |
| backend | NodeJS |
| persistence | MongoDB |
| cert and keys | OpenSSL |
| frontend | React Native |

## How to launch

backend
```bash
$ node server.js
```

frontend
```bash
$ npx react-native run-android --verbose
```
